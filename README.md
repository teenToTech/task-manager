📝 Task Manager (Python CLI Project)

A simple command-line based Task Manager built using Python.  
It allows users to add, view, delete, and mark tasks as completed.  
All tasks are stored permanently using a JSON file.

---

## 🚀 Features

- Add new tasks with priority
- View all tasks
- Delete tasks by index
- Mark tasks as completed
- Data saved in `tasks.json` file

---

## 🛠️ Tech Stack

- Python
- JSON (for data storage)
- File Handling

---

## 📁 Project Structure


Task Manager/

├── task_manager.py
└── tasks.json (auto-created)

1. Make sure Python is installed
2. Run the file:

```bash
